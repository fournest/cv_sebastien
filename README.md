# CV - Sébastien FOURNEST

CV en ligne au format A4 pour Sébastien FOURNEST, Développeur Fullstack.

## Description

Ce projet est un CV interactif et imprimable, optimisé pour l'export en PDF au format A4. Il présente :

- **Profil** : Développeur Fullstack avec 20 ans d'expérience opérationnelle
- **Compétences techniques** : Symfony, PHP, Node.js, Express, JavaScript, MySQL, MongoDB, Git
- **Projets** : Application de Monitoring GitHub, Plateforme de Gestion Locative
- **Expériences** : Stage Développeur, Chauffeur de Taxi, Logistique & Management
- **Formation** : Titre Pro DWWM, Permis Taxi

## Fichiers

- `index.html` - Structure HTML du CV
- `style.css` - Styles CSS, design responsive et optimisé pour l'impression A4

## Utilisation

1. Ouvrir `index.html` dans un navigateur
2. Pour imprimer/exporter en PDF :
   - Utiliser `Ctrl + P` (ou `Cmd + P` sur Mac)
   - Sélectionner "Enregistrer en PDF"
   - Les paramètres de marges sont préconfigurés pour le format A4

## Design

- **Sidebar** (30%) : Informations personnelles, coordonnées, compétences, langues
- **Contenu principal** (70%) : Résumé, projets, expériences, formations
- **Couleurs** : Design épuré avec fond bleu-gris et accents bleus
- **Format** : Strictement A4 (21cm x 29.7cm)
